Para rodar o código da DFT, é necessário ter as seguintes bibliotecas:

Tkinter
numpy, versão 1.14.5
matplotlib, versão 2.2.2

A instalação do Tkinter depende do seu SO, mas para instala-lo no Ubuntu, rode o seguinte comando:
sudo apt-get install python3-tk

As outras duas bibliotecas podem ser instaladas através do comando:
pip3 install -r requirementes.txt

Para executar o código, entre no arquivo main.py e altere as variáves X e Y de acordo com as amostras da sua função. Em seguida execute o comando:
python3 main.py
